<h1 align="center">👁️ Vision Pulse</h1>

<p align="center">
  <strong>Vision Pulse</strong> は、AI を活用したリアルタイム物体検出 Web アプリケーションです。<br>
  Web カメラの映像を使用して、物体の検出・セグメンテーション・カウントをリアルタイムで実行します。<br>
  また、人やコップなどの特定の物体をフィルタリングして表示することもできます。
</p>

![VisionPulseDemo](https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/VisionPulseDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/assets/readme/features.gif" width="35"/> 主な機能

| 機能 | 説明 |
| ---- | ---- |
| 🎥 リアルタイム検出 | Web カメラを使用したリアルタイム物体検出 |
| 🧠 AI 搭載 | YOLOv8 セグメンテーションモデルによる高精度な検出 |
| 🎯 物体フィルタリング | 特定の物体のみを選択して検出可能 |
| 🔢 物体カウント | 検出された物体数をリアルタイムで表示 |
| ⚡ 高速処理 | スムーズで快適なパフォーマンスを実現 |
| 🌐 Web インターフェース | Flask を使用したシンプルなブラウザベース UI |

## <img src="https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/assets/readme/gettingStarted.gif" width="35"/> はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/VisionPulse.git
```

### 2️⃣ プロジェクトディレクトリへ移動する

```bash
cd VisionPulse
```

### 3️⃣ 依存関係をインストールする

```bash
pip install -r requirements.txt
```

### 4️⃣ アプリケーションを起動する

```bash
python app.py
```

### 5️⃣ ブラウザで開く

```text
http://127.0.0.1:5000/
```
