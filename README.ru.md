<h1 align="center">👁️ Vision Pulse</h1>

<p align="center">
  <strong>Vision Pulse</strong> это веб-приложение для обнаружения объектов в режиме реального времени, работающее на базе искусственного интеллекта.
  Оно использует поток с веб-камеры для обнаружения, сегментации и подсчёта объектов в реальном времени, позволяя пользователям фильтровать определённые объекты, такие как люди, кружки и многие другие.
</p>

## <img src="https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/assets/readme/features.gif" width="35"/> Возможности

| Возможность | Описание |
| ------------ | -------- |
| 🎥 Обнаружение в реальном времени | Обнаружение объектов через веб-камеру в режиме реального времени |
| 🧠 Искусственный интеллект | Использует модель сегментации YOLOv8 для высокой точности |
| 🎯 Фильтрация объектов | Фильтрация обнаружения по выбранным объектам |
| 🔢 Подсчёт объектов | Отображает количество обнаруженных объектов в режиме реального времени |
| ⚡ Высокая производительность | Оптимизировано для плавной и быстрой работы |
| 🌐 Веб-интерфейс | Простой интерфейс, работающий через Flask |

## <img src="https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/assets/readme/gettingStarted.gif" width="35"/> Начало работы

### 1️⃣ Клонируйте репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/VisionPulse.git
````

### 2️⃣ Перейдите в папку проекта

```bash
cd VisionPulse
```

### 3️⃣ Установите зависимости

```bash
pip install -r requirements.txt
```

### 4️⃣ Запустите приложение

```bash
python app.py
```

### 5️⃣ Откройте в браузере

```text
http://127.0.0.1:5000/
```
