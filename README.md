# 🧿 Vision Pulse

**Vision Pulse** is a real-time object detection web application powered by AI.
It uses a webcam feed to detect, segment, and count objects dynamically, allowing users to filter specific objects like people, cup, and more.

![VisionPulseDemo](https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/VisionPulseDemo.png)

## ✨ Features

| Feature             | Description                                                 |
| ------------------- | ----------------------------------------------------------- |
| 🎥 Live Detection   | Real-time object detection using webcam                     |
| 🧠 AI-Powered       | Uses YOLOv8 segmentation model for accurate results         |
| 🎯 Object Filtering | Filter detection by specific objects                        |
| 🔢 Object Counting  | Displays count of detected objects in real-time             |
| ⚡ Fast Processing   | Optimized for smooth and responsive performance             |
| 🌐 Web Interface    | Simple browser-based UI using Flask                         |

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash id="k3m9x2"

```

### 2️⃣ Navigate to the Project

```bash
cd VisionPulse
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the App

```bash

```

