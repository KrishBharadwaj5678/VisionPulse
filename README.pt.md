<h1 align="center">👁️ Vision Pulse</h1>

<p align="center">
  <strong>Vision Pulse</strong> é uma aplicação web de detecção de objetos em tempo real alimentada por Inteligência Artificial.
  Ela utiliza o feed da webcam para detectar, segmentar e contar objetos dinamicamente, permitindo que os usuários filtrem objetos específicos, como pessoas, copos e muitos outros.
</p>

![VisionPulseDemo](https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/VisionPulseDemo.png)

## <img src="https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/assets/readme/features.gif" width="35"/> Funcionalidades

| Funcionalidade | Descrição |
| -------------- | --------- |
| 🎥 Detecção em Tempo Real | Detecta objetos em tempo real usando a webcam |
| 🧠 Alimentado por IA | Utiliza o modelo de segmentação YOLOv8 para resultados precisos |
| 🎯 Filtro de Objetos | Permite filtrar a detecção por objetos específicos |
| 🔢 Contagem de Objetos | Exibe a quantidade de objetos detectados em tempo real |
| ⚡ Alto Desempenho | Otimizado para uma execução rápida e responsiva |
| 🌐 Interface Web | Interface simples baseada em navegador utilizando Flask |

## <img src="https://github.com/KrishBharadwaj5678/VisionPulse/raw/main/assets/readme/gettingStarted.gif" width="35"/> Como Começar

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/KrishBharadwaj5678/VisionPulse.git
```

### 2️⃣ Acesse a pasta do projeto

```bash
cd VisionPulse
```

### 3️⃣ Instale as dependências

```bash
pip install -r requirements.txt
```

### 4️⃣ Execute a aplicação

```bash
python app.py
```

### 5️⃣ Abra no navegador

```text
http://127.0.0.1:5000/
```
